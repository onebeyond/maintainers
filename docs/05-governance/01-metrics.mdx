# Metrics

We use Metrics to collect data about the performance of our open source activity and help us to shape the future by providing vital information to take decisions based on data and measure the sucess of our Initiatives. 

## Metrics we use

We use the following metrics:
- Community engagement with [Cauldron](https://cauldron.io/). Here you can see [the report](https://cauldron.io/project/7192)
- Code Quality with [Code Climate](https://codeclimate.com/). Each repo has a badge with the current status
- Security Maintainers best practices with [OpenSSF Scorecard](https://github.com/ossf/scorecard). Each repo has a badge with the current status
